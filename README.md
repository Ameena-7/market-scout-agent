AI-powered competitor change tracking agent for verified weekly technical updates.

# Market Scout Agent

## Problem
Consulting teams struggle to identify verified recent technical updates.

## Solution
AI agent that searches, verifies and summarizes only new technical changes from last 7 days.

## Architecture
5-layer modular architecture.

## Tech Stack
- Python
- OpenAI API
- Serper API
- BeautifulSoup
- Streamlit

## Setup Instructions
1. Clone repo
2. Install requirements
3. Run app.py
